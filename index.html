Implementación del método matricial

Esta herramienta está desarrollada para optimizar modelos a través del método matricial, de acuerdo a los siguientes pasos:

  • Debe elegir si desea maximizar o minimizar el modelo
  • Ingresar luego cuántas variables tiene el modelo e ingresarlas
  • Ingresar los coeficientes de la variable objetivo
  • Ingresar el número de restricciones e ingresar los coeficientes de cada restricción por cada variable junto con cada variable independiente.
  • Al terminar de ingresar los datos, el sistema le mostrará lo siguiente:
  • El Vector de variables
  • La Matriz de coeficientes de las resctricciones (Matriz A)
  • El Vector de recursos
  • El Vector de coeficientes de la función objetivo.
